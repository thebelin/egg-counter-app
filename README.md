## An app demonstrating how to build an ionic app for counting eggs

Usage:
````
$ ionic start ionic-egg-counter https://github.com/thebelin/egg-counter-app
$ cd ionic-egg-counter
$ ionic platform add android
$ ionic build android
$ ionic run android
````
